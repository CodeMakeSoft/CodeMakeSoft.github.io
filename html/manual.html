MANUAL DE INSTALACIÓN

Este es el manual y las configuraciones recomendadas para ejecutar el proyecto en tu sistema.

DEPENDENCIAS DEL SISTEMA

  • Sistema Operativo: Windows 10+/ Linux/ macOS
  • Composer: Mínimo 2.8.8
  • Node.js: Mínimo 22.14.0
  • NPM: Mínimo 10.9.2
  • Git: Recomendado 2.49.0
  • XAMPP: Recomendado 8.2.12-0 (VS16)
    • Apache: Recomendado 2.4.x
    • MySQL/MariaDB: 10.4 o superior
    • PHP: 8.2.12 o superior
  • VSCode: Editor recomendado
  • Espacio en disco: Mínimo 400MB

ENLACES PARA DESCARGAR

  • Composer: Descargar Composer
  • Node.js: Descargar Node.js

  • Git: Descargar Git

    • Apache: Incluido en XAMPP

INSTALACIÓN POR CONSOLA (LINUX/MACOS)

Composer

Linux & MacOs

# Instalar Composer
curl -sS https://getcomposer.org/installer | php
# Mover composer.phar a una ubicación accesible globalmente
sudo mv composer.phar /usr/local/bin/composer
# Verificar la instalación
composer --version

Node.js y npm

Linux (Debian/Ubuntu)

# Actualizar el índice de paquetes y instalar Node.js
sudo apt update
sudo apt install nodejs npm

# Verificar la instalación
node --version
npm --version

MacOs

# Instalar Node.js con Homebrew:
brew install node

# Verificar la instalación
node --version
npm --version

Git

Linux (Debian/Ubuntu)

# Instalar Git
sudo apt update
sudo apt install git

# Verificar la instalación
git --version

MacOs

# Instalar Git
brew install git

# Verificar la instalación
git --version

VSCode

Linux (Debian/Ubuntu)

# Instalar VSCode
sudo apt update
sudo apt install code

# Verificar la instalación
code --version

MacOs

# Instalar VSCode
brew install --cask visual-studio-code

# Verificar la instalación
code --version

XAMPP

Descarga y ejecuta el .run para Linux o el .dmg para MacOs (asegurate de darle permisos de ejecución) desde aquí.

Desactivar antivirus (únicamente en Windows)

El antivirus puede bloquear archivos o conexiones de Composer, confundiéndolos con falsos positivos (cree que son virus ya que descarga y descomprime muchos archivos al mismo tiempo), causando errores. Desactiva la protección en tiempo real temporalmente en Defender o cualquier otro antivirus.

INSTALACIÓN

Descarga el siguiente script y guárdalo en htdocs:  install.sh

 

Posiciónate en el directorio htdocs y ejecuta una shell con interprete de Bash.

Windows

Ejecuta Git Bash (se instalo junto con Git) y añade estos comandos:

# Muevete a la ruta: C:\xampp\htdocs
cd C:/xampp/htdocs

# Ejecuta el script
./install.sh

Linux/MacOs

Ejecuta la terminal nativa o la de tu preferencia y añade estos comandos:

Linux

# Muevete a la ruta: /opt/lampp/htdocs
cd /opt/lampp/htdocs

# Ejecuta el script
./install.sh

MacOs

# Muevete a la ruta: /Applications/XAMPP/htdocs
cd /Applications/XAMPP/htdocs

# Ejecuta el script
./install.sh

Instalación finalizada (Linux/MacOs/Windows)

Cuando haya concluido la instalación ejecuta en tu terminal:

code P_UPPStore/blue_app

Esto abrira el proyecto en VSCode (recuerda que puede ser el editor de tu preferencia).

EJECUCIÓN DEL PROYECTO

Figura 4: Proyecto en VSCode.

Ejecuta dos terminales sobre blue_app, en la primera añade:

npm run dev

En la segunda añade:

php artisan serve

Esto iniciara el servidor de desarrollo y el servidor de Laravel, respectivamente.

Abre tu navegador y dirígete a la siguiente dirección:

http://localhost:8000
