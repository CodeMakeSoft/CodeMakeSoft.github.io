MANUAL DE INSTALACIÓN

Este es el manual y las configuraciones recomendadas para ejecutar el proyecto en tu sistema.

Requisitos Previos

  • Sistema Operativo: Windows 10+/Linux/macOS
  • Node.js: Versión 16.x o superior
  • Git: Para clonar el repositorio
  • Espacio en disco: Mínimo 500MB

Instalación

# Clonar repositorio
        git clone https://github.com/tu-usuario/tu-proyecto.git
        cd tu-proyecto
# Instalar dependencias
        npm install
        # ó
        yarn install

Configuración

Crea un archivo .env en la raíz del proyecto con:

API_KEY=tu_clave_aqui
        PORT=3000
        DB_URL=mongodb://localhost:27017/tu_db

Ejecución

# Modo desarrollo
        npm run dev
        
        # Modo producción
        npm start

Solución de Problemas

Error: Puerto en uso

Solución: Libera el puerto o cambia la configuración:

# Linux/macOS
        kill -9 $(lsof -ti:3000)
        
        # Windows
        taskkill /F /PID
